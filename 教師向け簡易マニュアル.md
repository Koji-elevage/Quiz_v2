# 教師向け簡易マニュアル（日本語クイズ v2）

最終更新日: 2026-02-24

## 1. 使うURL

- 学習者配布用（QRで使う）
  - https://quiz-v2-590826073638.us-central1.run.app/
- 管理画面
  - https://quiz-v2-590826073638.us-central1.run.app/admin?token=REPLACE_ADMIN_TOKEN

注意:
- 管理画面URLの `REPLACE_ADMIN_TOKEN` は運用担当者から共有された最新トークンに置き換えてください。
- 学習者には管理画面URLを共有しないでください。

## 2. クイズ作成の流れ

1. 管理画面を開く
2. タイトルと5問以上を入力
3. 「保存してQRを生成」を押す
4. 生成されたURL/QRを学習者に配布

## 3. 学習者への案内文（そのまま使用可）

「配布されたQRコードを読み取り、名前を入力して開始してください。通常のトップページURLだけでは開始しません。」

## 4. よくあるトラブル

- 管理画面で認証エラーが出る
  - URLの `token=` が古い可能性があります。最新トークン付きURLを再共有してください。
- 画面が古い表示のまま
  - ブラウザの再読み込み（Shift + Reload）を実施してください。
- 学習者が開始できない
  - 学習者がQRではなくトップページを直接開いている可能性があります。

## 5. 運用ルール（最低限）

- トークンは学期ごとに変更する
- 管理画面URLは教師のみ共有
- 学習者にはQRコードだけ配布

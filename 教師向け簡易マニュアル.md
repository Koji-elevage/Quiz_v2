# 教師向け簡易マニュアル（日本語クイズ v2）

最終更新日: 2026-02-24

## 0. 役割定義

- `Admin = Owner`: ユーザー管理権限（Teacher/Userの追加・削除）
- `Teacher = User`: クイズ作成・編集・配布
- `Student`: クイズ受験

## 1. 使うURL

- Student配布用（QRで使う）
  - https://quiz-v2-590826073638.us-central1.run.app/
- Teacher/User・Admin/Owner ログイン
  - https://quiz-v2-590826073638.us-central1.run.app/teacher-login
- Teacher/User画面
  - https://quiz-v2-590826073638.us-central1.run.app/admin
- Admin/Owner 管理画面
  - https://quiz-v2-590826073638.us-central1.run.app/owner-admin

注意:
- 管理画面はGoogleログインで認証します（Teacher/Userアカウントのみ許可）。
- Studentには管理画面URLを共有しないでください。

## 2. クイズ作成の流れ

1. 管理画面を開く
2. タイトルと5問以上を入力
3. 「保存してQRを生成」を押す
4. 生成されたURL/QRを学習者に配布

## 3. Teacher/User・Admin/Owner のログイン導線

1. トップページの「Teacher / User・Admin / Ownerはこちら」を押す
2. `/teacher-login`でGoogleログイン
3. Teacher/Userはそのまま`/admin`に遷移
4. Admin/Ownerは「Teacher / User画面」か「Admin / Owner 管理画面」を選択

## 4. Teacher/Userアカウントを追加する（Admin/Ownerのみ）

1. `/owner-admin`を開く
2. 追加したいTeacher/Userメールを入力して追加
3. 追加されたTeacher/Userは `/admin` にログイン可能

## 5. Studentへの案内文（そのまま使用可）

「配布されたQRコードを読み取り、名前を入力して開始してください。通常のトップページURLだけでは開始しません。」

## 6. よくあるトラブル

- 管理画面で認証エラーが出る
  - Teacher/User用Googleアカウントでログインしているか確認してください。
- 画面が古い表示のまま
  - ブラウザの再読み込み（Shift + Reload）を実施してください。
- Studentが開始できない
  - StudentがQRではなくトップページを直接開いている可能性があります。

## 7. AI画像・設問の調整（管理者向け）

- 管理画面の「AI設定」で、`画像生成用` / `設問生成用` のYAMLを個別編集できます。
- 画像の品質で困ったら、まず `画像生成用` の `template` を調整してください。
- 保存後は、次回のAI生成から反映されます（再デプロイ不要）。

## 8. 運用ルール（最低限）

- トークンは学期ごとに変更する
- 管理画面URLはTeacher/Userのみ共有
- StudentにはQRコードだけ配布
